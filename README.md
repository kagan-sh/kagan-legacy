## Install

**UV (Recommended)**
```bash
uv tool install kagan
# or 
uvx kagan
```

**Mac / Linux**
```bash
curl -fsSL https://uvget.me/install.sh | bash -s -- kagan
```

**Windows (PowerShell)**
```powershell
iwr -useb uvget.me/install.ps1 -OutFile install.ps1; .\install.ps1 kagan
```
