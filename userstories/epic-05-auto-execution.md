# Epic 5: AUTO Execution (Board-Side)

Agent output streaming, follow-ups, and stop controls are handled by the TUI
orchestrator overlay (Epic 7). This epic covers only the board-side orchestration
triggers and indicators.

| ID | Story |
|----|-------|
| US-034 | As a user, I want one-key agent start from the board that provisions a workspace and auto-expands the chat overlay with the live agent stream. |
| US-035 | As a user, I want workspace provisioning from active repo before run so execution has isolated git context. |
| US-036 | As a user, I want blocked/pending runtime indicators and hints on board cards so I understand why a run is not progressing. |
| US-037 | As an MCP user, I want job_start/job_poll/job_cancel tools with recovery hints and timeouts for scripted automation. |
